# osTicket Helpdesk Lab

## Project Overview
I built a fully functional helpdesk ticketing system on Fedora Linux to learn IT support workflows.

## Technologies Used
- Fedora Linux
- Apache (HTTPD)
- MariaDB (MySQL)
- PHP 8.4
- osTicket v1.18.1

## What I Did
- Set up LAMP stack on Fedora
- Configured osTicket helpdesk system
- Created and resolved practice tickets for:
  - Printer troubleshooting
  - Password reset
  - Software installation
